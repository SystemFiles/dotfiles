# dotfiles
My personal dotfiles, managed with [`chezmoi`](https://github.com/twpayne/chezmoi).

## Installation

### Linux

```sh
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply SystemFiles
```

## Install location

```sh
~/.local/share/chezmoi
```
